# EgoPoseFormer v2

**Accurate Egocentric Human Motion Estimation for AR/VR**

[Paper](https://arxiv.org/abs/2603.04090) | [Project Page](https://robinkips.github.io/EgoPoseFormerv2/)

## CVPR 2026 Highlight

EgoPoseFormer v2 (EPFv2) is a transformer-based method for accurate and temporally-consistent egocentric 3D human motion estimation, paired with a scalable auto-labeling system that leverages large unlabeled datasets.

### Key Results (EgoBody3M benchmark)
- **4.02 cm** MPJPE (22.4% improvement over EgoBody3M)
- **0.8 ms** GPU latency
- **51.7%** reduction in temporal jitter vs. EgoPoseFormer
